## Color Coding System

### Availability Indicators
- ?? **Green** (>50%) - Good availability
- ?? **Yellow** (20-50%) - Limited availability  
- ?? **Red** (<20%) - Very limited availability
- ? **Gray** (0%) - Sold out

### Badge Colors
- ?? **Primary** (Purple gradient) - Genre, Active filters
- ?? **Danger** - Selling fast, Sold out
- ? **Dark** - Genre badge on image
- ? **Light/White** - Quick view button

## Interactive Elements

### Hover Effects
1. **Concert Card**
   - Lifts up (-8px)
   - Shadow increases
   - Shows quick view button
   - Image zooms in slightly

2. **Genre Pills**
   - Border color changes
   - Lifts up (-2px)
   - Shadow appears

3. **Buttons**
   - Background color changes
   - Slight scale effect

### Animations
- **Pulse Badge**: Pulsing animation for "Selling Fast"
- **Card Hover**: Smooth lift and shadow transition
- **Image Zoom**: Gentle zoom on hover
- **Genre Pills**: Bounce effect on hover

## State Management

### LocalStorage Keys
- `concertsView` - Stores user's preferred view (grid/list)

### URL Parameters
- `?search=query` - Search term
- `?genre=Rock` - Genre filter
- `?sortBy=price-low` - Sort option

### ViewBag Properties
- `ViewBag.Genres` - List of available genres
- `ViewBag.CurrentGenre` - Active genre filter
- `ViewBag.CurrentSearch` - Active search term
- `ViewBag.CurrentSort` - Active sort option
